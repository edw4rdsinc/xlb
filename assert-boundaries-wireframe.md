## **Design Specifications**

**Brand Colors:**
- Primary: Forest Green (hex code pending from Pauline)
- Secondary: Gold (hex code pending from Pauline)
- Neutrals: Warm earthy tones (beige, cream, soft browns)
- Accent: Deep green for CTAs

**Typography:**
- Headings: Modern serif font (warm, approachable, professional)
- Body Copy: Clean sans-serif (highly readable, accessible)
- Size Hierarchy: Clear distinction between H1, H2, H3, body

**Animation Strategy:**
- Hero Section: Smooth fade-in with slight scale ("pops at you" effect per meeting notes)
- Service Cards: Staggered fade-up on scroll
- Audience Selection: Subtle lift and shadow on hover
- Navigation: Smooth scroll to anchors
- Loading: Minimal, professional animations

**Image Requirements:**
- Professional portrait of Pauline (About page hero)
- Calm, welcoming therapy room (multiple uses)
- Diverse, culturally representative imagery
- Community and connection themes
- Natural textures and warm lighting
- Professional supervision/mentorship settings

**SimplePractice Integration Points:**
- Embedded scheduling widget on Contact page
- All "Schedule Consultation" CTAs link to widget
- "Request Consultation" buttons link to contact form
- Client portal link in footer

**Mobile Responsiveness:**
- Single column layout on mobile
- Touch-friendly buttons (min 44px tap targets)
- Simplified navigation menu
- Stacked service cards
- Side-by-side audience selection becomes stacked

---

**Project Scope:** $500 website (November 2025 agreement)
**Excludes:** Advanced automation, CRM setup, StoryBrand documentation process
**Includes:** Website design/development, StoryBrand messaging application, SimplePractice integration

This wireframe synthesizes the dual-path approach from the meeting with the comprehensive StoryBrand messaging from the proposal.
